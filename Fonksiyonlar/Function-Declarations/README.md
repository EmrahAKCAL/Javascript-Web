<br><br><b><i>function sum(x, y){</i></b><br>
  Yürütülecek kod bloğu<br>
<b><i>}</i></b><br>
Burada <b><i>sum </i></b> fonksiyona verilen isim, <b><i>x</i></b> ve <b><i>y</i></b> ise dışardan gelecek parametreleri tanımlar.
